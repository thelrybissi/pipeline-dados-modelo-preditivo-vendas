# pipeline-dados-modelo-preditivo-vendas
Construir um pipeline completo de ingestão, transformação, armazenamento e modelo preditivo.

Stack sugerido:

- Dados: CSV, API pública de vendas, ou dataset Kaggle

- ETL: PySpark + Delta Lake (Databricks)

- Orquestração: Airflow ou Azure Data Factory

- Modelagem: ML em Python (sklearn / LightGBM / XGBoost)

- Armazenamento: Azure Data Lake + Synapse

- Visualização: Power BI ou React dashboard

## O que vai demonstrar:

- Domínio de pipeline em batch

- Transformações em Spark + Delta Lake

- Deploy de modelo preditivo

- Integração de dados + dashboard



# Roadmap de 6 Meses – Engenharia de Dados + IA

## Mês 1 – Fundamentos de Dados
- Reforçar SQL avançado
- Estudar modelagem dimensional
- Aprender Python focado em ETL
- Praticar Pandas em datasets grandes

## Mês 2 – Pipelines e Armazenamento
- Aprender Spark (PySpark)
- Apache Airflow – conceitos e DAGs
- Data Lakes: Delta Lake / Iceberg
- Armazenamento: Azure Data Lake, S3

## Mês 3 – Cloud Data Engineering
- Azure Data Factory
- Azure Databricks
- Azure Synapse
- Infra como código (Terraform)

## Mês 4 – Machine Learning para Engenharia
- Scikit‑learn (prático)
- MLflow – rastreamento de modelos
- Feature Stores (Feast)
- Introdução a MLOps

## Mês 5 – LLMs e IA Generativa
- Fine‑tuning leve (LoRA)
- Construção de pipelines com LangChain
- Vetorização e RAG (Pinecone / Azure AI Search)
- Modelos open‑source (Llama, Mistral)

## Mês 6 – Projeto Final + Certificações
- Criar pipeline completo fim‑a‑fim
- Modelo em produção com MLOps
- Certificação: Azure Data Engineer ou AI‑900
